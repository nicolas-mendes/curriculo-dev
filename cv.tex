\section*{Projetos}
\begin{itemize}[noitemsep, nolistsep, leftmargin=*, itemsep=8pt]
    
    \item 
        \href{https://github.com/nicolas-mendes/JobSprout}{\raisebox{-0.05\height}\faGithub \textbf{ JobSprout - Plataforma de Anúncio de Vagas}} \\
        \href{https://jobsprout-lt91r.sevalla.app}{\raisebox{-0.05\height}\faGlobe \small \textit{Ver projeto online}} \\
        Desenvolvi uma plataforma de postagem de vagas completa com \textbf{PHP, Laravel, JavaScript, MySQL e Vite}, construindo uma UI responsiva e moderna com \textbf{Tailwind CSS} e componentes \textbf{Blade} reutilizáveis. \\ 
        Garanti a integridade e a segurança dos dados por meio da implementação de um sistema robusto de autenticação e autorização com Laravel, validação de entradas no back-end e proteção contra CSRF. A estrutura do banco de dados foi versionada com \textit{migrations} e \textit{seeders} de demonstração. \\ 
        Além disso, também lidei com a configuração de um ambiente de produção com configuração de \textit{database}, \textit{disks} e \textit{build}.

\newpage

    \item 
        \href{https://github.com/nicolas-mendes/type-hero}{\raisebox{-0.05\height}\faGithub \textbf{ Type Hero - Jogo Web de Digitação}} \\
        Desenvolvi um jogo web que combina mecânicas de RPG por turnos com desafios de digitação. \\
        O frontend foi construído com \textbf{JavaScript} e a engine \textbf{Phaser 3}, implementando lógica de combate em tempo real, manipulação de \textit{input} e animações.  \\
        Para o backend, arquitetei uma \textbf{API RESTful} utilizando \textbf{PHP Nativo}, responsável por gerenciar autenticação, sessões, progressão de ligas e rankings globais. 
        A persistência de dados utiliza \textbf{MySQL}, incluindo a criação de scripts automatizados para configuração e \textit{seeding} do banco de dados.

    \item 
        \href{https://github.com/nicolas-mendes/script-backup-linux}{\raisebox{-0.05\height}\faGithub \textbf{ Script de Backup Automatizado Linux}} \\
        Automatizei o processo de backup de diretórios locais, assegurando a integridade e a consistência dos dados com um sistema de verificação, através da criação de um script em Shell (Bash) com uma interface de menu interativa para o usuário, utilizando a função \textbf{`rsync`} para a sincronização eficiente e segura dos arquivos.
        
\end{itemize}
